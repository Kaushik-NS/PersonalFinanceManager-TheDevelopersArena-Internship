# **Personal Finance Manager**

</div>

---

## **Overview**

A Python console-based application to manage personal expenses with reporting and visualization.

---

## **Features**

- Add expenses  
- View all expenses  
- Category-wise summary  
- Monthly PDF reports  
- CSV storage  

---

## **How to Run**

1. Install **Python 3.x**
2. Run the application:
   ```bash
   python main.py
